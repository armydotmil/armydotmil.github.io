Upgrading Internet Explorer

Support guidance from Microsoft

Beginning January 12, 2016, only the current version of Internet Explorer available for a supported operating system will receive technical support and security updates. Microsoft recommends that customers running older versions of Internet Explorer upgrade to the most recent version, which is Internet Explorer 11 on Windows 7, Windows 8.1, and Windows 10.

Benefits of upgrading

  • Security - Hacks, viruses, malware, and other malicious security issues constantly evolve, and only browsers that evolve with modern preventative measures will be able to keep up. By opening yourself up to security risks, it not only affects you but can cause problems for the entire network you are connected to.
  • Modern web technology - HTML5, CSS3, and WebGL are examples of technology used in modern browsers. Without support for these current and future technologies, web pages utilizing them may look strange or even broken.
  • Speed - Image rendering, network optimization, caching, and many other factors play a role in how fast a page loads.
  • Features - New ways of organizing links and bookmarks, web monitoring tools, customization of allowed technologies, and many other features are only found on newer browsers.

Install a new browser

In many cases, you can simply download and install a new browser, and start using it right away. Be careful that you only download a new browser from an official source. If you do not have permission to install new programs on your computer, ask your network administrator to help. They should be able to install the browser of your choice or one from their own approved list.

Take a look at the list of modern browsers below. Each link will have information about the browser and how to download it, so you can research and compare which works best for you.

Automatically Update

Within the Windows operating system, there is an option to automatically update to the latest version. It can be found in the Control Panel, under Windows Update, and is possible to enable after clicking on Check for Updates. This option can be disabled by a network administrator as well, so be sure to contact them for assistance, if this is the case.
